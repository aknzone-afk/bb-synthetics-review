# Review Assistant Page

This page generates an AI-written review draft for B.B. Synthetics, copies it, and then opens the Google review page.

## What it does

- Generates a polished AI review draft for your saree manufacturing brand.
- Copies the selected review text.
- Opens your Google review page in a new tab.
- Falls back to the shared Supabase review pool if the AI endpoint is unavailable.

## Important limitation

Google does not provide a supported way to prefill and auto-post the public review text directly into the Google review form from a custom QR page. Customers will still need to paste or edit the copied text and then press `Post` on Google.
